# How To Install ?

Windows
<pre>
1.Download This Script
2.Install Nodejs in : nodejs.org
3.open install_windows.bat
</pre>

Linux
<pre>
1.apt-get install git -y
2.git clone https://github.com/FajarTheGGman/F-Chat
3.cd F-Chat
4.open install.sh
</pre>

# How To Run ?
<pre>
Just Type : node index.js
</pre>
